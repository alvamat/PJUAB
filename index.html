La botiga virtual

L'objectiu d'aquesta primera pràctica és disenyar un web site molt senzill per una botiga virtual. El web site ha de contenir una pàgina d'inici (descripció de la botiga, productes, etc.), una pàgina amb un llistat de productes i una pàgina amb un formulari que els usuaris poden fer servir per enviar les seves consultes/comentaris als administradors de la botiga.

Per facilitar-vos la feina, us proporcionem aquesta plantilla que haureu de modificar per tal de donar-li l'aspecte desitjat. Podeu canviar els colors, logo, tipus de navegació, etc. modificant les propietats corresponents del fitxer d'estil CSS.

Estructura de la pàgina

L'espai visible d'aquesta pàgina és divideix en 3 grans blocs: el header, el footer i el contenidor principal. Tant el header com el footer disposen d'un fons format per una imatge que es repeteix per tal d'ocupar tota l'amplada. El contenidor principal també el podem separar en dues parts ben diferenciades o dues columnes: el contingut i una barra lateral. La barra lateral conté una sèrie de links que ens permetran navegar per dintre del web site.

La disposició dels diferents elements seguint aquesta estructura la podem trobar integrament en el fitxer d'estil CSS twoColFixRtHdr.css. Podeu agafar aquestes propietats com a referència per afegir/eliminar/modificar elements de l'estructura.

Valoració final

Els aspectes que es valoraran en aquesta pràctica són, entre d'altres:

  • Ordenació dels elements completament separada del seu contingut (HTML vs CSS).
  • Coherència i estil adequats a la finalitat del web site (colors, fonts, navegació, etc.).
  • Llista de productes obtinguda de forma dinàmica utilitzant la base de dades MySQL que es posa a la vostra disposició.
  • Visualització dels productes (taula, grid, etc.).
  • Utilització d'elements d'estil CSS compatibles amb varis navegadors (evitar utilitzar instruccions específiques o que puguin produir comportaments inesperats en altres navegadors).

  • Disseny visualment atractiu (extra).
  • Web segura (extra)

Altres consideracions

La temàtica de la botiga virtual és lliure. Tot i així, heu de ser responsables i evitar temes que puguin afectar la integritat d'altres persones o puguin provocar un impacte emocional negatiu.

Qualsevol plagi detectat serà durament sancionat. La pràctica quedarà automàticament suspesa tant pel grup que copia com pel grup que es deixa copiar.

Per la realització d'aquesta pràctica es dedicaran dues sessions de dues hores al laboratori (Q5/2008). El codi del web site es podrà entregar fins 1h abans de la tercera sessió de pràctiques, sessió durant la qual el professor podrà plantejar algunes preguntes als membres del grup sobre la realització de la pràctica.
